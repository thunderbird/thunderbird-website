Thunderbird Notes

Version 55.0beta, first offered to channel users on July 15, 2017

Check out "What’s New" and "Known Issues" for this version of Thunderbird below. As always, you’re encouraged to tell us what you think, or file a bug in Bugzilla. If interested, please see the complete list of changes in this release.

These release notes apply to Thunderbird version 55 beta 2 released July 15th, 2017.

What’s New

  • changed

    In general, date/time display will follow the application locale. That means that for example the English-US version will display English-US date/time formats and the German version will display German date/time formats. Date/time display can be adjusted in the operating system settings as long as the language used in by the OS matches the language of the application. For example an English-US version will respect settings for any English locale, like English-United Kingdom, English-Australia or English-India. A French version will respect settings for any French locale, like France or Canada.

    Further improvements are scheduled for version 56 beta.
