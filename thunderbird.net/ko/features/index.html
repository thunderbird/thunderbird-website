Thunderbird 기능 개요

여러분의 필요에 맞춘 메일 프로그램 Thunderbird가 업그레이드가 더 쉬워졌습니다.

바로 사용 하기

계정 설정 마법사

지금까지 메일 계정을 만들 때 서버의 자세한 구성 정보를 모두 입력해야했습니다. 새로운 Thunderbird는 이름과 이메일 주소, 암호만 입력하면 Mozilla의 공급자 데이터베이스에서 자동으로 서버 설정 정보를 검색합니다.

원클릭 주소록

보내는 사람의 이메일 주소를 신속하면서도 쉽게 주소록에 등록 할 수 있게되었습니다. 받은 메시지에 ☆ 표 아이콘을 클릭하면 새로운 연락처를 추가 할 수 있습니다. 사진, 생일, 기타 연락처 정보를 편집하려면이 아이콘을 다시 클릭합니다.

첨부 알리미

메시지를 작성할 때 본문에 "첨부"등의 단어가 입력 될 때 알림 표시 줄을 표시하여 전송하기 전에 필요한 파일을 첨부 할 것을 알려 드립니다.

탭 및 검색 기능

탭

Thunderbird는 Firefox 탭 브라우징처럼 여러 메시지를 개별 탭에 열고 빠르게 전환하여 볼 수 있습니다. 예를 들어, 메시지에 답장을 쓰고있는 도중에 이전에 읽은 메시지에 다시 내용을 확인하고자 할 때,이 탭 기능을 사용하면 여러 메시지를 한 번에 동일한 창에서 열린 상태로 둘 수 있기 때문에 쉽게 볼 수 있습니다.

메시지를 두 번 클릭하거나 Enter 키를 누르면 해당 메시지가 다른 탭에서 열립니다. 메시지나 폴더를 마우스 오른쪽 단추로 백그라운드 탭으로 열 수 있습니다.

Thunderbird를 종료 할 때 열려있는 탭 상태가 기억되어, 다음 번 시작 시 복원됩니다. 탭 표시 줄의 오른쪽 끝에 열린 탭의 제목을 볼 수있는 드롭 다운 목록도 많은 탭을 열 때 쉽게 전환 할 수 있습니다.

빠른 필터 도구 모음

빠른 필터 도구 모음은 폴더의 메시지를 빠르게 필터링 할 수 있는 기능입니다. 임의의 검색어를 입력하기 시작하면 즉시 결과가 표시됩니다. 읽지 않은 메시지 태그 발신자가 주소록에 등록되어 있는지 등 조건에서도 추출 할 수 있습니다. 또한 "핀"아이콘을 클릭하면 폴더보기를 전환해도 검색을 유지할 수 있습니다.

검색

Thunderbird의 검색 화면에 검색 도구와 타임 라인 도구가 포함되어 있어 원하는 메시지를 빠르고 정확하게 찾아 낼 수 있습니다. 모든 메시지는 데이터베이스에 등록되어 있기 때문에 빠른 검색이 가능합니다. 검색 결과는 다른 탭으로 표시되므로 다른 메시지도 쉽게 오갈 수 있습니다.

메시지 아카이브

나중에 필요하게 될 것 같은 메시지를 제거하지 않고 받은 편지함에서 지우고 싶은 경우, 아카이브 기능을 활용하면 됩니다. 새 아카이브 폴더에 메시지를 이동하고, 받은 편지함을 정리할 수 있습니다.

아카이브 버튼 또는 'A'를 누르면 선택된 메시지가 보관됩니다.

이벤트 로그 관리

"이벤트 로그 관리"창에는 송수신 등의 기록이 모두 기록됩니다. 언제 메시지를 다운로드했는지 등을 추측 할 필요가 더 이상 없습니다. 하나의 화면에서 자신의 계정 상태를 파악할 수 있습니다.

맞춤형 이메일 경험

디자인 선택

Personas에서 마음에 드는 테마를 선택 적용하면 Thunderbird의 디자인을 즉시 전환 할 수 있습니다. 영화, 패션, 풍경, 스포츠 등 수십만의 디자인 중에서 선택할 수 있습니다. 아이콘 등 세부 사항까지 디자인을 변경할 수있는 테마도 있습니다.

스마트 폴더

스마트 폴더 모드에서는 별도의 사서함이 함께 표시되어 여러 개의 이메일 계정을 효율적으로 관리 할 수 있습니다. 예를 들어, 받은 편지함을 선택하면 각 계정을 개별적으로 열지 않고 모든 계정의 새 메시지를 표시 할 수 있습니다.

부가 기능 관리자

부가 기능 관리자를 열기 만하면 Thunderbird에서 직접 확장 기능을 검색하고 설치할 수있게되었습니다. 부가 기능 웹 사이트를 방문 할 필요가 더 이상 없습니다. 어떤 기능을 사용하면 좋을지 모르겠다 때도 사용자의 평가, 추천, 설명, 사진 등을보고 선택할 수 있습니다.

강력해진 이메일 보안

강력한 개인 정보 보호

Thunderbird는 HTML 메시지에 포함 된 원격 이미지로드를 차단하고 사용자의 개인 정보를 보호합니다.

피싱 경고 기능

Thunderbird는 수신 된 메시지에 개인 정보 나 기밀 정보를 속여 취하려는 피싱의 위험성이 의심되는 경우 경고를 표시합니다. 또한 추가 대책으로 메시지에 표시되는 URL과 실제로 열리는 웹 사이트가 다른 의심스러운 링크가 클릭 된 경우에도 브라우저에서 해당 사이트가 열리기 전에 경고를 표시합니다.

자동 업데이트

Thunderbird 업데이트 시스템은 사용자가 최신 버전을 실행하고 있는지 확인하고 보안 업데이트가 제공 될 때 알려줍니다. 보안 업데이트는 작지만 (일반적으로 200KB - 700KB) 필요한 항목 만 제공하고 보안 업데이트를 다운로드하여 설치하는 작업을 신속하게 수행합니다. 자동 업데이트 시스템은 40 이상의 언어로 Windows, macOS 및 Linux에서 Thunderbird에 대한 업데이트를 제공합니다.

학습형 스팸 필터

이전부터 호평을 받고있는 스팸 방지 기능은 다양한 스팸 수법에 대응할 수 있도록 만들어져 있습니다. 모든 새 메시지는 학습형 스팸 필터로 해석됩니다. 사용자가 스팸 메일의 판정을 할 때마다, 그 내용이 축적되어 자동 판별의 정확도가 향상하기 때문에 더 많은 시간을 중요한 메시지 교환에 할애할 수 있습니다. 공급자와 사내 메일 서버의 스팸 필터를 병용하면받은 편지함에서 스팸 메일이 격감하는 것입니다.

오픈 소스 개발 프로세스

Thunderbird는 풍부한 경험이 개발자와 보안 전문가가 전세계에서 다수 참가하는 오픈 소스 개발 프로세스에 의해서 유지되고 있습니다. 투명한 환경 속에서 많은 전문가로부터 적극적인 협력을받는 것으로 보다 안전한 제품을 제공 할 수 있고, 만일 문제가보고 된 경우도 신속하게 수정 된 버전을 배포 할 수있는 체제를 갖추고 있습니다. 또한 동시에 뛰어난 타사의 보안 검증 · 평가 도구를 활용하여 전체적인 보안 수준을 향상시키고 있습니다.
